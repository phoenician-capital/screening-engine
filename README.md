## 🤖 The Two-Stage System

### Why Two Stages?

Most screening systems do this:

```
❌ TRADITIONAL (expensive & slow)
1000 companies → [One agent evaluates all] 
                → $1000 cost
                → 7200+ seconds runtime
                → Too many false positives
```

**Our system works smarter:**

```
✅ PHOENICIAN (intelligent & efficient)
1000 companies
    ↓
[SELECTION TEAM: Fast & Strict]
• Rejects obvious misfits (rules-based)
• Keeps only quality candidates
• Cost: $10 | Time: 150 seconds
    ↓
50 companies ← Only the best pass through
    ↓
[SCORING TEAM: Thorough & Intelligent]
• Evaluates each carefully (AI-powered)
• 6 scoring dimensions
• Cost: $50 | Time: 400 seconds
    ↓
✨ 50 ranked recommendations
```

**Total Cost: $60** (vs $1000 without selection)  
**Time Saved: 90%** ⚡  
**Quality Improved:** Better signal-to-noise ratio

## 🧠 The Selection Team (Stage 1)

Five specialized agents that say "YES" or "NO" to each company:

### Agent 1: 🔧 **The Filter Agent**
**Asks:** "Do the basic financials make sense?"

Checks:
- ✓ Gross margin > 30%? (Is it profitable?)
- ✓ ROIC > 10%? (Is capital used efficiently?)
- ✓ Debt levels reasonable? (Not overleveraged?)
- ✓ Revenue & profit positive? (Real business?)

**Rejects:** ~40% (fail basic financial tests)

---

### Agent 2: 🏢 **The Business Model Agent**
**Asks:** "Can we understand what this company does?"

Uses AI to judge:
- ✓ Simple, understandable model?
- ✗ Complex conglomerate? (Hard to analyze)
- ✗ Unclear business? (Red flag)

**Rejects:** ~20% (unclear business models)

---

### Agent 3: 👔 **The Founder Agent**
**Asks:** "Do insiders have skin in the game?"

Checks:
- ✓ Founder or CEO owns ≥ 5% of company?
- ✓ Recent insider buying activity?
- ✗ All insiders sold recently? (They don't believe)

**Rejects:** ~30% (no insider alignment)

---

### Agent 4: 📈 **The Growth Agent**
**Asks:** "Is this growth real or just acquisitions?"

Analyzes:
- ✓ Organic growth > 3%? (Real market expansion)
- ✗ Growth only via acquisitions? (Not sustainable)
- ✗ Acquisition spending > FCF? (Overpaying)

**Rejects:** ~25% (acquisition-heavy growth)

---

### Agent 5: 🚩 **The Red Flag Agent**
**Asks:** "Any learned red flags from past analyst feedback?"

Checks:
- ✓ Previous similar companies: what happened?
- ✗ Does this match a pattern you rejected before?
- ✗ Did you learn to avoid this metric level?

**Rejects:** ~35% (learned red flags)

---

## 💡 The Scoring Team (Stage 2)

### Agent 1: 🧠 **The AI Analyst Agent**

Deep analysis across 6 dimensions:

| Dimension | What's Checked | Example |
|-----------|---|---|
| **Business Quality** | Real competitive advantage? | Strong brand moat? |
| **Unit Economics** | Core metrics healthy? | Sustainable margins? |
| **Capital Returns** | Shareholder returns high? | ROIC > 15%? |
| **Growth Quality** | Sustainable growth? | Organic or acquisitions? |
| **Balance Sheet** | Financially strong? | Reasonable debt, growing cash? |
| **Phoenician Fit** | Match our criteria? | Founder-led, proven model? |

**Output:** Fit Score (0-100)
- 0-40: Poor fit
- 40-60: Potential
- 60-75: **Research Now** (good!)
- 75-90: Very Strong
- 90+: Exceptional

---

### Agent 2: ⚠️ **The Risk Scorer**

Assesses downside risks:

**Evaluates:**
- How much can margins fall before unprofitable?
- What if competition increases?
- Is business vulnerable to disruption?
- How stable are the financials?

**Output:** Risk Score (0-100)
- 0-15: Very safe
- 15-30: Normal risk
- 30-50: Higher risk (manageable)
- 50-70: Significant downside
- 70+: Too risky

---

System: [still forgetting]

Next month...

Analyst: "Why do you keep recommending these??"
System: [has no memory whatsoever]

## 🎓 Key Concepts Explained Simply

### What is "Confidence"?

**Confidence** = How sure we are a learned pattern is correct.

```
New pattern from your feedback:
├─ Starts at: 70% (somewhat sure)
├─ You see similar company #1: You agree! ✓
│  Confidence: 70% → 75%
├─ You see similar company #2: You agree! ✓
│  Confidence: 75% → 80%
├─ You see similar company #3: You agree! ✓
│  Confidence: 80% → 85%
│
└─ When confidence > 75%:
   Pattern automatically applies to ALL companies
   (System doesn't need your approval anymore)
```

### What is "Learned Pattern"?

**Learned Pattern** = A rule created from your feedback.

```
PATTERN EXAMPLE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Rule Name: High Buyback Ratio Red Flag

Trigger: When stock_repurchase / FCF > 0.8x
Action: Automatically reject company in selection
Origin: Your feedback on AXON company
Confidence: 85% (you've validated it 5 times)
Expires: 30 days without updates (auto-decay)

How it works:
  New company arrives with 2.5x buyback/FCF
  System checks rules
  "This exceeds 0.8x threshold!"
  Company automatically rejected
  Analyst never sees it (saves $1)
```

### What is "Two-Stage System"?

**Two-Stage** = Different teams for different jobs.

```
STAGE 1: SELECTION (Fast & Strict)
├─ Goal: Filter obvious misfits quickly
├─ Cost: Cheap (mostly rule-based)
├─ Speed: Very fast (150 sec for 1000)
├─ Rejects: 95% of companies
└─ Output: 50 best candidates

STAGE 2: SCORING (Thorough & Smart)
├─ Goal: Deep analysis of good candidates
├─ Cost: Expensive (AI-powered)
├─ Speed: Slower (but only on 50 companies)
├─ Accepts: Detailed evaluation
└─ Output: Ranked recommendations

WHY IT WORKS:
You don't spend expensive scoring resources
on obvious rejects. Selection weeds them out first.
```

### What is "Server-Sent Events"?

**SSE** = Real-time updates from server to your browser.

```
OLD WAY (polling):
└─ Browser asks "Are we done yet?" every 5 seconds
   └─ Waits, then asks again
   └─ Waits, then asks again
   └─ Result: Feels slow, jittery

NEW WAY (SSE):
└─ Server automatically sends updates:
   ├─ "Discovery complete: 1000 candidates"
   ├─ "Selection complete: 50 candidates"
   └─ "Scoring complete: Results ready"
   └─ Result: Smooth, beautiful, real-time
```
